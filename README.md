# Spoon-chatall
Script alternatif untuk mengirimkan pesan ke semua room dari rank 1 hingga batas rank yang telah ditentukan

## Installation (Instalasi)
Download terlebih dahulu termux di playstore ya (Gratis)

```bash
pkg install python -y
pkg install git -y
pkg install nano -y
git clone https://github.com/diverglovsky/chatall.git
```

## Usage (Cara pakai)

```bash
cd chatall
python main.py
```

## Edit config
```bash
cd chatall
nano config.ini
```
Edit bagian nomor, password, pesan , dan max

```
[Spoon]
nomor = 083142399531
password = qweasdzxc1234
pesan = pagi gaes, join sini script gratis . ramaikan kuy https://t.me/joinchat/UPFZEKRg6jTVdpCv
max = 90
```
- nomor dan password untuk login akun  
- pesan ya pesan  
- max adalah batasan rank , dari rank 1 hingga rank max yang telah ditentukan
